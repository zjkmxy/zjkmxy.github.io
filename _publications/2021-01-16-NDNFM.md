Up to now, configuring NDN for use on end hosts has generally been difficult due to the absence of graphical configuration interfaces. To improve the usability of NDN, we
have developed the NDN Forwarder Manager (NDN-FM).
NDN-FM provides a graphical interface for users to manage a local instance of the NDN Forwarding Daemon (NFD),
allowing them to monitor the status of the forwarder; create, update, and delete faces and routes; manage certificates;
and run basic NDN debugging tools. NDN-FM can also be
used to manage other NDN packet forwarders that support
the NFD Management Protocol. In this report, we describe
the implementation of NDN-FM, discuss the rationale for
the design choices that were made during its development,
and demonstrate the use cases of NDN-FM from the perspective of system administrators and end users to illustrate
NDN-FM’s improvements to NDN usability.
